# 3tier-aws-assignment
# 3-Tier AWS Web Application

## Project Overview

This project demonstrates deployment of a secure 3-tier web application on AWS.

Technology Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Cloud: AWS

AWS Services

- Elastic Beanstalk
- Application Load Balancer
- VPC
- Public & Private Subnets
- Internet Gateway
- NAT Gateway
- Security Groups
- CloudWatch
- S3 (Frontend)
- CloudFront

Architecture

User
↓
CloudFront
↓
Amazon S3
↓
Application Load Balancer
↓
Elastic Beanstalk
↓
MongoDB Atlas
